Q: Can I still create Stock Entries from BOM?
A: Yes! This module adds an ALTERNATIVE workflow via Production List.

Q: Can I edit Work Order items after submit?
A: No, but you can edit them BEFORE submit. After submit, create a PL.

Q: Can I create multiple Production Lists from one Work Order?
A: Yes! Each PL tracks its own items and balance.

Q: What happens if I cancel a Stock Entry?
A: Production List automatically recalculates and updates balance/status.

Q: Can I transfer partial quantities?
A: Yes! Use qty_dict parameter in make_stock_entry_from_production_list()
